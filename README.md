# 🛒 eSuperMart Sales Data Analysis Project

## 📌 Project Description

The eSuperMart Sales Data Analysis project focuses on analyzing retail sales data to uncover key business insights related to revenue, customer purchasing behavior, and product performance.

Using Power BI, the dataset was cleaned, transformed, and visualized to build an interactive dashboard. The dashboard provides a comprehensive overview of sales trends, profit distribution, and category-wise performance, helping stakeholders make informed business decisions.

This project demonstrates strong skills in data visualization, data cleaning, and extracting actionable insights from real-world datasets.

---

## 🎯 Objectives

* Analyze overall sales and profit performance
* Identify top-performing products and categories
* Understand customer purchasing patterns
* Track regional sales trends and performance

---

## 🛠 Tools & Technologies

* Power BI (Dashboard & Visualization)
* Data Cleaning & Transformation
* Data Analysis

---

## 📊 Key Insights

* Identified top-selling product categories contributing to maximum revenue
* Analyzed profit distribution across different regions
* Observed seasonal trends and sales fluctuations
* Highlighted underperforming segments for improvement
